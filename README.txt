Code Ladder - A simple program to teach other Visual Basic programming
======================================================================
 
Code Ladder is a simple educational tool. It presents simple programming
puzzles and evaluates entered solutions.  It uses the Microsoft.Reflection
library to compile and assemble code in memory on-the-fly.


Contact & Bug Reports
---------------------

To report bugs, suggest features, ask a question, get the latest version,
or help out, visit http://code.google.com/p/code-ladder/.
